# AWS Database Lab

Projeto prático para estudo de banco de dados relacional utilizando PostgreSQL,
com foco em modelagem, consultas SQL e futura integração com AWS RDS.

## Tecnologias utilizadas
- PostgreSQL
- SQL
- GitHub
- AWS (RDS - Free Tier)

## Estrutura do projeto

aws-database-lab/
├── sql/
│   ├── create_tables.sql
│   └── queries.sql
└── README.md

## Objetivo
- Criar um banco de dados relacional simples
- Praticar comandos DDL e DML
- Executar consultas com JOIN, GROUP BY e funções agregadas
- Preparar ambiente para uso na AWS (RDS)
